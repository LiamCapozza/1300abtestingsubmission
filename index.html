Detailed A/B Testing Report - Liam Capozza

Project Intro

This A/B testing project tries to evaluate the user interactions of two different versions of a Doctor's appt. form. By comparing the two versions, A and B, we are looking for which version has better user interaction and efficiency.

Webpage Modifications

I changed the layout of the appointments for Version B. I was aiming for better visibility and distinction between buttons and texts which would hopefully result in fewer misclicks and a faster experience.

Metrics and Hypotheses

1. Misclick Rate

Null Hypothesis: Version A of the form will have a very similar average misclick rate as Version B. I believe we will reject this hypothesis since Version A had buttons and text that are hard to see and differentiate from one another.

Alternative Hypothesis: Version B will have a lower average misclick rate than Version A. I believe this will be found true since the changes I made make it easier to see and differentiate different appointments from one another.

2. Time on Page

Null Hypothesis:Version A will have a very similar average time on the page as Version B. I believe we will reject this hypothesis because Version A is confusing in how to exactly book an appointment, so the user will have to spend more time figuring out what to press.

Alternative Hypothesis: Version B will have a lower time on page than Version A. I believe we will prove this hypothesis because of the highlight and border around each different appointment, making it easy to tell which buttons belong to which appointment

3. Mouse Move Distance

Null Hypothesis: Version A will have a very similar average mouse move distance as Version B. I believe we will reject this because I think mouse move distance will be correlated with the amount of user confusion, as they might go to click on option but then realize that they need to click another.

Alternative Hypothesis: Version B will have a lower average mouse move distance than Version A. I believe this because the changes made to Version B allow for less confusion when finding an appointment, leading to fewer mistakes or misclicks, therefore less distance moved by the mouse.

Statistical Analysis Results

Misclick Rate Analysis

We will use a chi squared test because we are checking how often the users of version A misclicked versus how often the users of version B misclicked. Since a user can misclick or not misclick (true or false), this is categorical data.

Chi-Squared: 8.710, P-Value: .003.

We reject the null hypothesis, since with a p-value less than or equal to 0.05, we can say that there is a statistically significant difference between the two datasets, specifically that there were a lot less misclicks in Version B than Version A due to the high chi-squared value.

Time on Page Analysis

We will use a one-tailed t test since the time on page is a continuous variable and we are looking to see that version B has a lower average time than version A.

T-Score: -4.512, P-Value: .0001, Version A Avg: 25865ms, Version B Avg: 8503ms.

We reject the null hypothesis, since with a p-value less than or equal to 0.05, we can confidently say that there is a significant statistical difference between the two versions, with the T-score implying that Version B is lower than Version A. We can see this is the case with A’s average time on page being 25865ms while B’s is 8503ms. A is over 3 times as long as B is.

Mouse Move Distance Analysis

We will use a one-tailed t test again since distance moved is a continuous variable and we are hoping to find that version B has a lower average distance than version A.

T-Score: -3.372, P-Value: .002, Version A Avg: 7337, Version B Avg: 2683.

We reject the null hypothesis, since with a p-value less than or equal to 0.05, we can confidently say that there is a significant statistical difference between the two versions, with the T-score implying that Version B is lower than Version A. We can see this is the case with A’s average mouse move distance being 7337 while B’s is 2683. A is almost 3 times as long as B is.

Summary and Conclusions

Since we are able to reject the null hypothesis on all three chosen metrics, we can say with high levels of confidence that Version B creates a more streamlined and user-friendly experience compared to Version A. Version B has an almost perfect no misclick rate while Version A has almost 50% misclick rate. In both time on page and mouse move distance, Version A's mean was about 3x as long as Version B's mean.
